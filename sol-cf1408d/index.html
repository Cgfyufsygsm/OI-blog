CF1408D Searchlights

Oct 24, 2021
阅读时长: 1 分钟

Description

给定 \(n\) 个点 \((a_i, b_i)\)，和 \(m\) 个探照灯 \((c_i, d_i)\)。探照灯 \(j\) 能看到点 \(i\) 当且仅当 \(a_i\le c_j\land b_i\le d_j\)。一次操作可以将所有的 \((a_i, b_i)\) 平移一个单位（沿 \(x\) 轴或 \(y\) 轴），问最少几次操作后能使得所有的点都不被探照灯发现。

\(1\le n,m\le 2000\)，\(0\le a_i,b_i,c_i,d_i\le 10^6\)。

Solution

我们不妨分开考虑 \(x\) 轴和 \(y\) 轴，令 \(f_i\) 为 \(x\) 轴方向上移动 \(i\) 格后，\(y\) 轴方向上需要的最小移动次数。

然后就 \(O(nm)\) 枚举每个点和探照灯，对 \(f\) 取后缀 \(\max\) 的同时统计答案即可。

Implementation

const int maxn = 2005;
int a[maxn], b[maxn], c[maxn], d[maxn], n, m, f[1000005];

int main() {
    read(n), read(m);
    FOR(i, 1, n) read(a[i]), read(b[i]);
    FOR(i, 1, m) read(c[i]), read(d[i]);
    FOR(i, 1, n) FOR(j, 1, m) if (c[j] >= a[i]) chkmax(f[c[j] - a[i]], d[j] - b[i] + 1);
    int ans = 1000001, cur = 0;
    DEC(i, 1000001, 0) chkmax(cur, f[i]), chkmin(ans, cur + i);
    print(ans, '\n');
    return output(), 0;
}
