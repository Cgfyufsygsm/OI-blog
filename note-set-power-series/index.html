集合幂级数胡扯

阅读时长: 11 分钟

前言

好难啊。NOI 前随缘来胡扯点东西。

由于是写给自己看的所以可能写的很暴躁简略，如果看不懂可以去 Ref 部分看一下大佬们的博客（

定义

一种定义在集合上的形式幂级数。

为了方便，定义记号

  • \(2^U\) 表示 \(U\) 的幂集，即 \(U\) 的所有子集构成的集合。
  • 定义 \(U\) 为全集 \(\{1,2,\cdots, n\}\)，其中 \(n=|U|\)，注意到我们这里只是为了区分集合的 \(n\) 个元素。
  • 下文若无特殊说明，都有 \(S\in 2^{U}\)。

域 \(F\) 上的集合幂级数为映射 \(f:2^U\to F\)，其中对于每个 \(S\subseteq U\)，\(f\) 都有对应取值 \(f_S\in F\)。即我们相当于把全集的幂集到 \(F\) 建立了一个映射。

类比形式幂级数，我们在每个 \(f_S\) 后面添加占位符 \(x^S\)，即 \(cx^S\) 表示 \(f\) 会将 \(S\) 映射到 \(c\)。因此，我们一般如是表示： \[ F(x) = \sum_{S\subseteq U} f_Sx^S \]

加法/乘法

形式幂级数的加减法（对应系数相加减）和乘法（加法卷积，FFT/NTT）大家都已熟知，而集合幂级数同样可以定义加减法和乘法。

加减法与形式幂级数类似，都为对应系数相加减，两幂级数对应的全集需要相同。

而乘法的定义就略微复杂。我们希望乘法有关于加法的分配律，此时就需要定义一下指数的运算 \(\oplus\)，要求

  • 交换律：\(S\oplus T = T\oplus S\)；
  • 结合律：\((A\oplus B)\oplus C = A\oplus (B\oplus C)\)；
  • 单位元：\(A\oplus \varnothing = A\)。

则 \[ H(x) = F(x)G(x) = \sum_{S\in 2^U}\sum_{T\in 2^U}(f_Sg_T)x^{S\oplus T} \] 而常见的运算符包括交，并和对称差，下文将逐个阐述。

集合并卷积

定义

\[ h_S = \sum_{L\in 2^U}\sum_{R\in 2^U}[L\cup R = S]f_Lg_R \]

从位运算的角度来讲，这就是按位或卷积： \[ h_i = \sum_{j\mid k = i}f_jg_k \] 暴力求解 \(O(4^n)\)，劣。

莫比乌斯变换

我们可以使用将 \(F(x)\) 和 \(G(x)\) 分别进行莫比乌斯变换后求解。具体地我们定义集合幂级数 \(F(x)\) 的莫比乌斯变换 \(\hat{F}(x)\) 满足 \[ \hat f_S = \sum_{T\subseteq S}f_T \] 然后将 \(\hat F(x)\) 和 \(\hat G(x)\) 按位相乘即可，考虑证明： \[ \begin{aligned} \hat h_S &= \hat f_S\hat g_S\\ &= \sum_{L\subseteq S}f_L\sum_{R\subseteq S} g_R\\ &= \sum_{L\subseteq S}\sum_{R\subseteq S}f_Lg_R\\ &= \sum_{L\subseteq S}\sum_{R\subseteq S}[(L\cup R)\subseteq S] f_Lg_R \end{aligned} \] 最后需要把 \(\hat H(x)\) 变换回 \(H(x)\)，关于 FMT 是如何完成的见下文。

这个过程和 FFT 啥的很像，都是将其点值化然后直接将对应点值相乘，再逆变换回去。

高维前缀和

发现如果我们用二进制数的思想理解集合，则莫比乌斯变换就相当于一个高维前缀和，其中每个二进制位都是一维。

具体地，枚举二进制位，然后对于下标 \(i\) 进行加减操作之类的。

void FMT(int f[], int c) {
    for (int i = 1; i < (1 << n); i <<= 1)
        for (int j = 0; j < (1 << n); ++j)
            if (i & j) f[j] += c * f[j ^ i];
    return; // 此处没有考虑取模之类的问题，只是示意代码。
}

传进去的 \(c = 1\) 表示是莫比乌斯变换，\(-1\) 表示其逆变换。

枚举二进制位这一个过程相当于就是在枚举做高维前缀和的维度。

FMT 的性质

  • FMT 是线性变换，下文讲 FWT_OR 的时候详细说明。
  • \(\prod F_i(x) = \operatorname{IFMT}(\prod\operatorname{FMT}(F_i(x))\)。其中左边的 $$ 是集合并卷积，右边的是对应点值相乘。所以求若干集合幂级数的并卷积的时候可以直接先 FMT 好然后一波带走，不需要像形式幂级数那样 DFT，两个乘起来之后就得 IDFT。因为两个长 \(n\) 的多项式卷起来后长 \(2n\)，而两个定义在相同全集上的集合幂级数乘起来后还是定义在同一全集上。

集合对称差卷积

定义

定义两个集合 \(A\) 和 \(B\) 的对称差为 \(A\Delta B = \complement_{A\cup B}(A\cap B)\)，相当于异或运算。 \[ h_S = \sum_{L\in 2^U}\sum_{R\in 2^U}[L\Delta R = S]f_Lg_R \] 从位运算卷积的角度理解其就是异或卷积。 \[ h_k = \sum_{i\oplus j = k}f_ig_j \] 其中 \(\oplus\) 为异或运算。

快速沃尔什变换/FWT

将 \(F(x)\) 和 \(G(x)\) 分别进行沃尔什变换之后对应位置相乘，再沃尔什逆变换回去即可。

下文具体阐述，有点抽象。

FWT

构造

设 \(\operatorname{FWT}(A)\) 为幂级数 \(A\) 变换之后得到的幂级数。令 \(C = A * B\) 为 \(A\) 与 \(B\) 位运算卷积后的结果。则需要满足： \[ A*B = C\iff \operatorname{FWT}(A)\cdot\operatorname{FWT}(B) = \operatorname{FWT}(C) \] FFT 是一个线性变换，相当于乘上一个范德蒙矩阵。考虑对 FWT 构造一个类似的转移矩阵。

设 \(c(i, j)\) 为变换系数，即 \(A_j\) 对 \(\operatorname{FWT}(A)_i\) 产生的贡献，则 \[ \operatorname{FWT}(A)_i = \sum_{j = 0}^{n - 1}c(i, j)A_j \] 根据 \(\operatorname{FWT}(A)_i\cdot\operatorname{FWT}(B)_i=\operatorname{FWT}(C)_i\)，发现 \[ \begin{aligned} \left(\sum_{j = 0}^{n - 1}c(i, j)A_j\right)\left(\sum_{j = 0}^{n - 1}c(i, j)B_j\right) &= \sum_{j = 0}^{n - 1}c(i, j)C_j\\ \sum_{j = 0}^{n - 1}\sum_{k = 0}^{n - 1}c(i,j)c(i, k)A_jB_k &= \sum_{j = 0}^{n -1}c(i, j)C_j \end{aligned} \] 而 \(A*B = C\)， \[ \begin{aligned} \sum_{j = 0}^{n - 1}c(i, j)C_j &= \sum_{j = 0}^{n - 1}c(i,j)\sum_{t_1\oplus t_2 = j}A_{t_1}B_{t_2}\\ \sum_{j = 0}^{n - 1}\sum_{k = 0}^{n - 1}c(i,j)c(i, k)A_jB_k &= \sum_{j = 0}^{n - 1}c(i,j)\sum_{t_1\oplus t_2 = j}A_{t_1}B_{t_2}\\ &= \sum_{j = 0}^{n - 1}\sum_{t_1\oplus t_2 = j}A_{t_1}B_{t_2}c(i, j)\\ &= \sum_{t_1 = 0}^{n - 1}\sum_{t_2 = 0}^{n - 1}A_{t_1}B_{t_2}c(i, t_1\oplus t_2) \end{aligned} \] 对比以下左右两边，发现 \[ c(i, j)c(i, k) = c(i, j\oplus k) \] 同时，位运算是具有按位独立性的，所以知道了所有的 \(c(x, y)\)（\(x, y\in \{0, 1\}\)）之后就可以求出所有的 \(c(i, j)\)，具体地，把 \(i\) 和 \(j\) 的第 \(k\) 位称为 \(i_k\) 和 \(j_k\)，则 \(c(i, j) = \prod c(i_k, j_k)\)，证明显然。

现在考虑 \(\operatorname{FWT}(A)_i\) 的计算。前面说过，其等于 \[ \sum_{j = 0}^{n - 1}c(i, j)A_j \] 然后计算就是 \(O(n^2)\) 的，貌似没什么卵用。

接下来假定 \(n = 2^m\)，然后按位折半： \[ \begin{aligned} \operatorname{FWT}(A)_i &= \sum_{j = 0}^{n - 1}c(i, j)A_j\\ &= \sum_{j = 0}^{n / 2 - 1}c(i, j)A_j + \sum_{j = n/2}^{n - 1}c(i, j)A_j\\ &= c(i_0, 0)\sum_{j = 0}^{n / 2 - 1}c(i', j')A_j + c(i_0, 1)\sum_{j = n / 2}^{n - 1}c(i', j')A_j\\ \end{aligned} \] 然后就会惊奇的发现：问题的规模得到了减半。设 \(A\) 的前一半为 \(A_0\)，后一半为 \(A_1\)，则 \[ \begin{aligned} \operatorname{FWT}(A)_i &=c(0, 0)\operatorname{FWT}(A_0)_i + c(0, 1)\operatorname{FWT}(A_1)_i&i_0 = 0, i\in[0, n / 2)\\ \operatorname{FWT}(A)_{i + n/2}&= c(1, 0)\operatorname{FWT}(A_0)_i + c(1, 1)\operatorname{FWT}(A_1)_i&i_0 = 1, i\in[0, n / 2) \end{aligned} \] 分析一下时间复杂度：\(T(n) = T(n / 2) + O(n) = O(n\log n)\)。逆变换也就是 \(c\) 矩阵的逆，具体下文。

具体卷积

对于不同的位运算 \(\oplus\)，构造相应的 \(c\) 矩阵。将 \[ \begin{bmatrix} c(0, 0)&c(0, 1)\\ c(1, 0)&c(1, 1) \end{bmatrix} \] 称为位矩阵。

OR 卷积

\(c(i, j)c(i, k) = c(i, j\lor k)\)。

\(c(i, j)c(i, j) = c(i, j)\)，故 \(c(i, j)\in\{0, 1\}\)。 \[ \begin{cases} c(1, 0)c(1, 1) = c(1, 1)\\ c(0, 1)c(0, 0) = c(0, 1) \end{cases} \] 合法的矩阵有 \(\begin{bmatrix}1 & 1 \\ 1 & 0\end{bmatrix}\) 和 \(\begin{bmatrix}1 & 0\\1 & 1\end{bmatrix}\)。

第二个的意义为 \(c(i, j) = [i\land j = j]\)，\(\operatorname{FWT}(A)_i = \displaystyle\sum_{j = 0}^{n - 1}c(i, j) A_j = \sum_{j\in i}A_j\)。相当于就是子集求和，并且其逆阵的形式很对称，所以使用第二个矩阵。 \[ \begin{aligned} \operatorname{FWT}(A)_i &=\operatorname{FWT}(A_0)_i &i_0 = 0, i\in[0, n / 2)\\ \operatorname{FWT}(A)_{i + n/2}&= \operatorname{FWT}(A_0)_i + \operatorname{FWT}(A_1)_i&i_0 = 1, i\in[0, n / 2) \end{aligned} \]

void OR(modint *f) {
    for (int o = 2, k = 1; o <= n; o <<= 1, k <<= 1)
        for (int i = 0; i < n; i += o)
            for (int j = 0; j < k; ++j)
                f[i + j + k] += f[i + j];
    return;
}

\[ \begin{bmatrix} 1 & 0\\ 1 & 1 \end{bmatrix}^{-1} = \begin{bmatrix} 1 & 0\\ -1 & 1 \end{bmatrix} \]

\[ \begin{aligned} \operatorname{IFWT}(A)_i &=\operatorname{IFWT}(A_0)_i &i_0 = 0, i\in[0, n / 2)\\ \operatorname{IFWT}(A)_{i + n/2}&= \operatorname{IFWT}(A_0)_i - \operatorname{IFWT}(A_1)_i&i_0 = 1, i\in[0, n / 2) \end{aligned} \]

所以就可以写出如下的代码：

void OR(modint *f, modint x = 1) {
    for (int o = 2, k = 1; o <= n; o <<= 1, k <<= 1)
        for (int i = 0; i < n; i += o)
            for (int j = 0; j < k; ++j)
                f[i + j + k] += f[i + j] * x;
    return;
}

AND 卷积

\(c(i, j)c(j, k) = c(i,j\land k)\)，类似地，\(c(i,j)\in\{0,1\}\)。 \[ \begin{cases} c(0, 0)c(0, 1) = c(0, 0)\\ c(1, 0)c(1, 1) = c(1, 0)\\ \end{cases} \] 合法的矩阵有 \(\begin{bmatrix}1 & 1\\ 0 & 1\end{bmatrix}\) 和 \(\begin{bmatrix}0 & 1\\1 & 1\end{bmatrix}\)。第一种的意义为 \(c(i, j) = [i\land j = i]\)，\(\operatorname{FWT}(A)_i = \displaystyle\sum_{j=0}^{n - 1}c(i,j)A_j = \sum_{i\in j}A_j\)，相当于超集求和，且求逆完的形式比较对偶，所以使用第一种： \[ \begin{aligned} \operatorname{FWT}(A)_i &=\operatorname{FWT}(A_0)_i + \operatorname{FWT}(A_1)_i &i_0 = 0, i\in[0, n / 2)\\ \operatorname{FWT}(A)_{i + n/2}&=\operatorname{FWT}(A_1)_i &i_0 = 1, i\in[0, n / 2) \end{aligned} \]

\[ \begin{bmatrix} 1 & 1\\0 & 1 \end{bmatrix}^{-1} = \begin{bmatrix} 1 & -1\\ 0 & 1 \end{bmatrix} \]

\[ \begin{aligned} \operatorname{IFWT}(A)_i &=\operatorname{IFWT}(A_0)_i - \operatorname{IFWT}(A_1)_i &i_0 = 0, i\in[0, n / 2)\\ \operatorname{IFWT}(A)_{i + n/2}&=\operatorname{IFWT}(A_1)_i &i_0 = 1, i\in[0, n / 2) \end{aligned} \]

void AND(modint *f, modint x = 1) {
    for (int o = 2, k = 1; o <= n; o <<= 1, k <<= 1)
        for (int i = 0; i < n; i += o)
            for (int j = 0; j < k; ++j)
                f[i + j] += f[i + j + k] * x;
    return;
}

XOR 卷积

\(c(i, j)c(i, k) = c(i, j\operatorname{xor} k)\)。

合法的有 \(\begin{bmatrix}1 & 1\\ 1 & -1\end{bmatrix}\) 和 \(\begin{bmatrix}1 & 1 \\ -1 & 1\end{bmatrix}\)。第二种求完逆形式不太好，选用第一种。 \[ \begin{aligned} \operatorname{FWT}(A)_i &=\operatorname{FWT}(A_0)_i + \operatorname{FWT}(A_1)_i&i_0 = 0, i\in[0, n / 2)\\ \operatorname{FWT}(A)_{i + n/2}&= \operatorname{FWT}(A_0)_i -\operatorname{FWT}(A_1)_i&i_0 = 1, i\in[0, n / 2) \end{aligned} \]

\[ \begin{bmatrix} 1 & 1 \\ 1 & -1 \end{bmatrix}^{-1} = \begin{bmatrix} 1/ 2 & 1/ 2\\ 1/2 & -1/2 \end{bmatrix} \]

\[ \begin{aligned} \operatorname{IFWT}(A)_i &=\frac 1 2\operatorname{IFWT}(A_0)_i + \frac 1 2\operatorname{IFWT}(A_1)_i&i_0 = 0, i\in[0, n / 2)\\ \operatorname{IFWT}(A)_{i + n/2}&= \frac 1 2\operatorname{IFWT}(A_0)_i -\frac 1 2\operatorname{IFWT}(A_1)_i&i_0 = 1, i\in[0, n / 2) \end{aligned} \]

void XOR(modint *f, modint x = 1) {
    for (int o = 2, k = 1; o <= n; o <<= 1, k <<= 1)
        for (int i = 0; i < n; i += o)
            for (int j = 0; j < k; ++j) {
                f[i + j] += f[i + j + k];
                f[i + j + k] = f[i + j] - f[i + j + k] - f[i + j + k];
                f[i + j] *= x, f[i + j + k] *= x;
            }
    return;
}

代码实现

洛谷 P4717 的代码实现。

const int maxn = 1 << 19;

int n;
modint A[maxn], B[maxn], a[maxn], b[maxn];

void OR(modint *f, modint x = 1) {
    for (int o = 2, k = 1; o <= n; o <<= 1, k <<= 1)
        for (int i = 0; i < n; i += o)
            for (int j = 0; j < k; ++j)
                f[i + j + k] += f[i + j] * x;
    return;
}

void AND(modint *f, modint x = 1) {
    for (int o = 2, k = 1; o <= n; o <<= 1, k <<= 1)
        for (int i = 0; i < n; i += o)
            for (int j = 0; j < k; ++j)
                f[i + j] += f[i + j + k] * x;
    return;
}

void XOR(modint *f, modint x = 1) {
    for (int o = 2, k = 1; o <= n; o <<= 1, k <<= 1)
        for (int i = 0; i < n; i += o)
            for (int j = 0; j < k; ++j) {
                f[i + j] += f[i + j + k];
                f[i + j + k] = f[i + j] - f[i + j + k] - f[i + j + k];
                f[i + j] *= x, f[i + j + k] *= x;
            }
    return;
}

void mul(modint *f, modint *g) {
    FOR(i, 0, n) f[i] *= g[i];
    return;
}

int main() {
    read(n);
    n = 1 << n;
    FOR(i, 0, n - 1) read(A[i]);
    FOR(i, 0, n - 1) read(B[i]);
    
    cpy(A, a), cpy(B, b), OR(a, 1), OR(b, 1), mul(a, b), OR(a, MOD - 1);
    FOR(i, 0, n - 1) print(a[i]), putchar(' '); putchar('\n');

    cpy(A, a), cpy(B, b), AND(a, 1), AND(b, 1), mul(a, b), AND(a, MOD - 1);
    FOR(i, 0, n - 1) print(a[i]), putchar(' '); putchar('\n');

    cpy(A, a), cpy(B, b), XOR(a, 1), XOR(b, 1), mul(a, b), XOR(a, (MOD + 1) / 2);
    FOR(i, 0, n - 1) print(a[i]), putchar(' '); putchar('\n');

    return output(), 0;
}

总结

整体的思路：

  1. 回顾 FFT，现在要求构造一种线性变换满足 \(\operatorname{FWT}(A)\cdot\operatorname{FWT}(B) = \operatorname{FWT}(C)\iff A*B = C\)。

  2. 考虑构造系数矩阵 \(c(i, j)\) 满足 \(\operatorname{FWT}(A)_i = \sum c(i, j)A_j\)。

  3. 发现 \(c(i, j)c(i, k) = c(i, j\oplus k)\)。

  4. 发现所有的 \(c(i, j)\) 可以由 \(c(x, y)(x,y\in\{0, 1\})\) 得到。

  5. 将 \(\operatorname{FWT}(A)_i\) 的式子折半拆开，发现变为两个子问题。 \[ \begin{aligned} \operatorname{FWT}(A)_i &=c(0, 0)\operatorname{FWT}(A_0)_i + c(0, 1)\operatorname{FWT}(A_1)_i&i_0 = 0, i\in[0, n / 2)\\ \operatorname{FWT}(A)_{i + n/2}&= c(1, 0)\operatorname{FWT}(A_0)_i + c(1, 1)\operatorname{FWT}(A_1)_i&i_0 = 1, i\in[0, n / 2) \end{aligned} \]

  6. 对于具体的位运算，构造出 \(\begin{bmatrix}c(0, 0)& c(0, 1)\\ c(1, 0) & c(1, 1) \end{bmatrix}\) 及其逆阵，尽量使得两种形式统一。

  7. 敲出迭代版代码。

性质

  • 由于 (i)FWT 的本质是使用一个矩阵来进行变换，所以其具有线性性，即。 \[ \operatorname{FWT}(A) + \operatorname{FWT}(B) = \operatorname{FWT}(A + B)\\ \operatorname{FWT}(cA) = c\operatorname{FWT}(A) \]

  • 发现 FMT 和 FWT_OR 是等价的，这也解释了上文为啥 FMT 具有线性性。

  • FWT_OR 等价于子集求和（高维前缀和），FWT_AND 等价于超集求和（高维后缀和）。

子集卷积

应用

会综合 SOSdp 的应用。

例题 \(1\)：ABC212H - Nim Counting

给定 \(n\)，\(k\) 和 \((A_1, \cdots, A_k)\)。可以如下安排石子：每堆石子数量必须是 \(A_i\) 之一，且石子堆数量小等于 \(n\)。问安排石子使得先手 Nim 必胜的方案数。

\(1\le n\le 2\times 10^5\)，\(1\le k, A_i\le 2^{17}\)，\(A_i\) 两两不同。

不难发现问题转化为求 \(\bigoplus_{i = 1}^m B_i\not=0\) 的 \(B_i\) 的方案数，其中 \(m\) 为石子个数。

先考虑固定 \(m\)。则构造序列 \(C\) 使得 \(C_i = [\exist j, A_j = i]\)，定义卷积 \(*\) 为异或卷积，则 \(C * C * \cdots * C\)（\(m\) 个 \(C\)）的第 \(i\) 项系数即为异或和为 \(i\) 的方案数。这个过程是可以 FWT 做的，具体地，给 \(\operatorname{FWT}(C)_i\) 的每一项做一下快速幂再 IFWT 回去，把除了第 \(0\) 项之外的求个和，就得到了石子堆数为 \(m\) 的答案。

然后，不固定 \(m\) 怎么做呢？因为 FWT 是线性变换，而我们需要求的是 \(\operatorname{IFWT}(\operatorname{FWT}(C)_i + \operatorname{FWT}(C)_i^2 + \operatorname{FWT}(C)_i^3 + \cdots \operatorname{FWT}(C)_i^n)\)，所以定义 \(C'_i = \operatorname{FWT}(C)_i\)，然后求出 \(D_i = C_i' + C_i'^2 + \cdots + C_i'^n\) 再求 \(\operatorname{IFWT}(D)\) 把除了第 \(0\) 项外的系数累加即可。

例题 \(2\)：CF449D Jzzhu and Numbers

给定多重集 \(S\)，问从 \(S\) 中选子集使得子集内的数按位与结果为 \(0\) 的方案数对 \(10^9 + 7\) 取模的结果。\(n\le 10^6\)，\(0\le a_i\le 10^6\)。

不妨考虑二进制下数位的集合，令全集为 \(U\)，则可以对每个数 \(a_i\) 定义其集合幂级数 \(F_i(x) = x^{U} + x^{S_i}\)，然后 \([x^{\varnothing}]\prod F_i(x)\) 即为答案，其中 \(\prod\) 表示所有 \(F_i(x)\) 的集合与卷积。

朴素的 FWT_AND 可以做到 \(O(na\log a)\)，跑不过去。

但是每个 \(F_i(x)\) 只有 \(2\) 项不为 \(0\)。

所以 \(\operatorname{FWT}(F(x))_i = c(i, U) + c(i, a_i)\)，故其只能为 \(1\) 或 \(2\)。

现在需要求的就是对于每个 \(k\in[0, U]\)，求 \(\displaystyle\prod_{i = 0}^{n}(1 + c(k, a_i))\)。所以只要有 \(a_i\) 贡献的到的子集 \(k\)，其产生的贡献就是 \(2\)，知道了有多少个 \(a_i\) 能贡献到这个 \(k\) 之后一个快速幂就可以了。这个东西其实就是超集求和，可以使用 FWT_AND。具体地，设出一个序列 \(C\)，\(C_i\) 表示有多少个 \(a_k = i\)，然后对 \(C\) 做 FWT_AND 的结果为 \(C'\)，\(C'_k\) 就是能贡献到 \(k\) 的 \(a_i\) 的数量，再 \(C'_k = 2^{C'_k}\)，求出 \(\operatorname{IFWT}(C')\) 取出 \(\operatorname{IFWT}(C')_0\) 就是答案。

注意一下全 \(0\) 的情况，若把 \(lim\) 设为 \(1\) 则会多算一个空集，要特判。

const int maxn = 2e6 + 5;
modint f[maxn], pow[maxn];
int n;

void AND(modint *f, int lim, modint x = 1) {
    for (int o = 2, k = 1; o <= lim; o <<= 1, k <<= 1)
        for (int i = 0; i < lim; i += o)
            for (int j = 0; j < k; ++j)
                f[i + j] += f[i + j + k] * x;
    return;
}

int main() {
    read(n);
    FOR(i, 1, n) {
        int tmp; read(tmp);
        ++f[tmp];
    }
    pow[0] = 1;
    FOR(i, 1, n) pow[i] = 2 * pow[i - 1];
    int lim = 1 << 20;
    AND(f, lim, 1);
    FOR(i, 0, lim - 1) f[i] = pow[f[i].val];
    AND(f, lim, mod - 1);
    print(f[0]);
    return output(), 0;
}

例题 \(3\)：P8292 [省选联考 2022] 卡牌

给定 \(n\) 个整数 \(s_i\)，\(m\) 次询问。每次询问给出 \(c_i\) 个质数，问有多少种选法使得 \(\prod s_i\) 被这每个质数整除，答案模 \(998244353\)。\(1\le n\le 10^6\)，\(1\le s_i\le 2000\)，\(1\le m\le 1500\)，\(1\le \sum c_i\le 18000\)，\(2\le p_{i,j}\le 2000\)。

首先转化一下题意，给定 \(n\) 个质因子集合 \(S_i\)，有 \(m\) 次询问，每次询问给定一个质因子集合 \(T\)，问有多少种从这 \(n\) 个集合中选若干集合出来使得他们的并可以包含 \(T\) 的方案数。

然后发现对于 \(s_i\le 30\) 的部分分可以随便状压 dp，然后状态非常的像或卷积啥的，然后发现他就是一车集合幂级数的或卷积，具体地令 \(s_i\) 中 有 \(c_x\) 个 \(x\)，则 \(F_x(z) = z^{\varnothing} + (2^{c_x} - 1)z^{S_x}\)（加上个 \(z^{\varnothing}\) 是可以表示不选）。

那么对于 \(s_i\le 2000\)，回忆起有一道题叫做寿司晚宴，所以令 \(> \sqrt{2000}\) 的质数为大质数，而我们会注意到 \(\le \sqrt{2000}\) 的质数只有 $14 $ 个（下文令其集合为 \(P\)），非常的可以状压，并且大质数只会出现一次。这就是说，对于大质数 \(x,y\in T\)，我们是可以分开考虑的，反正不存在一个 \(S_i\) 能同时包含 \(x,y\)。

那先不考虑大质数，考虑 \(\in P\) 的所有质数的话，对所有的 \(x\) 预处理出 \(\operatorname{FWT}(F_x(z))\)，具体地我们发现原来的幂级数只有两项系数不为 \(0\)，所以包含了 \(S_x\) 的项的系数都为 \(2^{c_x}\)，否则为 \(1\)，都是 \(2\) 的次幂，所以可以只存次幂，预处理啥的很方便。

然后在每次询问的时候，考虑到 \(2^{|P|}\sum c_i\) 的复杂度可以接受，对每一个 \(\in T\) 但 \(\notin P\) 的质数 \(p\) 进行处理（\(\notin T,P\) 的质数不需要管）。不难注意到我们要去掉没有出现 \(p\) 的所有方案，那么对应地，我们得先除以满足 \(p\mid k\) 的所有 \(F_k(z)\)，然后再乘上 \(\displaystyle \prod_{p\mid k}F_k(z) - z^{\varnothing}\)（去掉不选的方案），最后的答案就是 \(\displaystyle\sum_{S\subseteq(T\cap P)}[x^S]M(z)\)，其中 \(M(z)\) 是最后得到的集合幂级数。

所以时间复杂度是 \(O((s_i + \sum c_i + m|P|)2^{|P|})\)，具体实现的时候，除法就考虑在 FWT 过后的点值 \(2\) 的次幂指数相减（都是 \(2\) 的次幂，逆元必然存在），注意提前预处理好所有的 \(F_x(z)\) 和 \(G_{p}(z) = \displaystyle\prod_{p\mid k} F_k(z)\)。

有一个细节就是大质数是只出现一种，不是一个，所以 \(43\) 可以是大质数（\(43\times 47>2000\)），但是要特判 \(43^2<2000\)。
