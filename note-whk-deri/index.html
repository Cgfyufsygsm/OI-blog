文化课导数笔记

Apr 27, 2022
阅读时长: 5 分钟

前言

基本定义全部不加证明。

放缩使用的不等式会浅证一下。

好 TM 难啊，文化课。

基本定义

定义 \(f(x)\) 在 \(x_0\) 处的导数 \(f'(x_0)\) 为 \[ \lim_{\Delta x\to 0}\frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x} \]

描述瞬时变化率/切线斜率。

常见导数公式

  • 常数函数： \[ (c)' = 0 \]

  • 多项式函数： \[ (x^n)' = nx^{n - 1} \] 对 \(n\in \mathbb R\) 成立。

  • 指数函数： \[ (a^x)' = a^x\ln a \] 特殊地，有 \[ (\mathrm e^x)' = \mathrm e^x \]

  • 对数函数： \[ (\log_ax)' = \frac{1}{x\ln a} \] 特殊地，有 \[ (\ln x)' = \frac{1}{x} \]

  • 三角函数： \[ \begin{aligned} (\sin x)' &= \cos x\\ (\cos x)' &= -\sin x\\ (\tan x)' &= \frac{1}{\cos^2 x} = \sec^2 x \end{aligned} \]

运算法则

  • 加减法： \[ f'(x)\pm g'(x) = (f(x) \pm g(x))' \]

  • 乘法： \[ (f(x)g(x))' = f(x)g'(x) + f'(x)g(x) \] 记忆：左导右不导 + 左不导右导。

  • 除法： \[ \left(\frac{f(x)}{g(x)}\right)' = \frac{f'(x)g(x) - f(x)g'(x)}{g^2(x)} \] 记忆：下面平方，上面减法，先导 \(f(x)\)。

  • 链式法则（复合函数求导）： \[ (f(g(x)))' = f'(g(x))g'(x) \] 记忆：“链式”。

常见的函数的导数：

  • \[ (x\ln x)' = 1 + \ln x \]

  • \[ (e^xf(x))' = e^x(f(x) + f'(x)) \]

  • \[ \left(\frac{f(x)}{e^x}\right)' = \frac{f'(x) - f(x)}{e^x} \]

此外，奇函数的导数为偶函数，偶函数的导数为奇函数（可以画图随便理解一下）。

函数的单调性与极值

  • \(f'(x)\) 在 \((a,b)\) 上 \(\ge 0\) \(\iff\) \(f(x)\) 在 \((a,b)\) 上单调增。
  • \(f'(x)\) 在 \((a,b)\) 上 \(\le 0\) \(\iff\) \(f(x)\) 在 \((a,b)\) 上单调减。

所以可以用导数来研究单调性一类的问题。求导没法解决的话可以考虑求二阶导，说不定就解决了。

极值指的是一个范围内的波峰/波谷，不要和最值 \(\max/\min\) 搞混。

\(f(x)\) 的极大值点为 \(a\) 当且仅当 \(f'(x)\) 在 \(a\) 前一段 \(\ge 0\)，\(f'(a) = 0\) 且 \(f'(x)\) 在 \(a\) 后一段 \(\le 0\)。极小值同理。

定义很傻逼，题目很阴间。

常见不等式

均值不等式

考虑给定一些数 \(a_1, a_2, \cdots, a_n\)，满足 \(a_i > 0\)，则若我们令：

  • 调和平均数 \(H_n = \displaystyle\frac{n}{\frac{1}{a_1} + \frac{1}{a_2} + \cdots \frac{1}{a_n}}\)
  • 几何平均数 \(G_n = \sqrt[n]{a_1\cdots a_n}\)
  • 代数平均数 \(A_n = \displaystyle\frac{a_1+\cdots +a_n}{n}\)
  • 平方平均数 \(Q_n = \displaystyle\sqrt{\frac{a_1^2 + \cdots a_n^2}{n}}\)

则有 \(H_n\le G_n\le A_n\le Q_n\)。等号成立条件均为 \(a_i\) 两两相等。

常见形式（下面假设出现的数均 \(>0\)）：

  • \(a+b\ge 2\sqrt{ab}\)；
  • \(ab\le \left(\displaystyle\frac{a + b}{2}\right)^2\)；
  • \(a^2 + b^2\ge 2ab\)。

不会证明。

对数均值不等式：\(\forall x_1 > x_2 > 0\)，有 \[ \sqrt{x_1x_2} < \frac{x_1 - x_2}{\ln x_1 - \ln x_2} < \frac{x_1 + x_2}{2} \]

证明：先考虑右侧，将双变量化为单变量：

需证 \(\displaystyle \frac{\frac{x_1}{x_2} - 1}{\frac{x_1}{x_2} + 1} < \frac{\ln\frac{x_1}{x_2}}{2}\)，浅换个元即为 \(\forall x > 1\)，\(\displaystyle\frac{2(x - 1)}{x + 1}< \ln x\)。构造函数 \(f(x) = \ln x - \dfrac{2(x + 1) - 4}{x + 1} = \ln x - \dfrac{4}{x + 1} - 2\)，发现 \(f(1) = 0\)，并且 \(f(x)\) 单调增，所以对于 \(x > 1\) 有 \(f(x) > 0\)。右侧证毕。

考虑左侧，需证 \(\ln\dfrac{x_1}{x_2} < \dfrac{x_1 - x_2}{\sqrt{x_1x_2}} = \sqrt{\dfrac{x_1}{x_2}} - \sqrt{\dfrac{x_2}{x_1}}\)，同样考虑换元，即 \(\forall x > 1\)，有 \(\ln x < \sqrt x - \sqrt{\dfrac 1x}\)。但是根号不太好，所以令 \(x:=\sqrt x > 1\)，换成 \(2\ln x < x - \dfrac 1x\)。同样构造函数即可证毕。

基本放缩

  • \(e^x\ge x+ 1\)。当且仅当 \(x = 0\) 时等号成立。

    证明：画图易证。

    考虑 \(f(x) = e^x - x - 1\)，求导 \(f'(x) = e^x - 1\)，易知 \(f'(x)\) 单调递增。故 \(x = 0\) 为 \(f(x)\) 极小值点，\(f(x)_{\min} = 0\ge 0\)，证毕。

  • \(\ln x\le x - 1\)。

    证明：画图易证。

    考虑 \(f(x) = \ln x - x + 1\)，求导 \(f'(x) = \dfrac1 x - x\)，易知 \(f'(x)\) 单调递减，故 \(x = 1\) 为 \(f(x)\) 极大值点，\(f(x)_{\max} = 0\le 0\)，证毕。

  • \(x\ge \ln(x + 1)\)。其实就是上面换了种写法。

  • \(e^x\ge ex\)。当且仅当 \(x = 1\) 时等号成立。

    证明：考虑 \(f(x) = e^x - ex\)，求导 \(f'(x) = e^x - e\)，易知 \(f'(x)\) 单调递增，\(x = 1\) 为 \(f(x)\) 极小值点，\(f(x)_{\min} = 0\ge 0\)，证毕。

麦克劳林级数

假定 \(x > 0\)，取 \(\exp(x) = \displaystyle\sum_{i\ge 0}\frac{x^i}{i!}\) 的前几项放缩一下就有 \[ e^x > \frac12 x^2 + x + 1 \] 证明：考虑构造函数 \(f(x) = e^x - \dfrac12 x^2 - x - 1\)，则 \(f'(x) = e^x - x - 1\)，\(f''(x) = e^x - 1\)，故对于 \((0,+\infin)\)，\(f'(x)\) 单调递增，而 \(f'(0) = 0\)，所以 \(f(x)\) 在 \((0,+\infin)\) 上单调递增，所以 \(f(x) > 0\)，证毕。

隐零点问题

引入

研究函数最值时，我们经常需要导函数的零点。

但是对于一些函数，无法显式求得其零点的值，例如 \(f(x) = e^x - \ln x\)，求完导是 \(f'(x) = e^x - \frac{1}{x}\)，会发现无法很方便求出其零点。

但是我们可以说明，\(f'(0.5) = \sqrt e - 2 < 0\)，\(f'(2) = e^2 - 0.5 > 0\)，由 \(f'(x)\) 的单调性和零点定理，其存在零点 \(x_0\in(0.5,2)\)。

代入变量法

例 \(1\)：证明：当 \(x>0\) 时，有 \(xe^x\ge x + \ln x + 1\)。

考虑令 \(f(x) = xe^x - x - \ln x - 1\)，则问题转化为证明 \(f(x)\) 在 \((0,+\infin)\) 上恒 \(\ge 0\)。

\(\displaystyle f'(x) = (x + 1)e^x - 1 - \frac 1x = (x + 1)(e^x - \frac 1x)\)，注意到 \((x + 1)>0\)。令 \(g(x) = e^x - \dfrac 1x\)，显然 \(g(x)\) 单调递增。

\(g(1/2) = \sqrt e - 2<0\)，\(g(2) = e^2 - \dfrac12 > 0\)，故存在 \(x_0\in(\dfrac12, 2)\) 满足 \(g(x_0) = 0\)。

于是有 \(e^{x_0} - \dfrac {1}{x_0} = 0\)，即 \(x_0e^{x_0} = 1\) 且 \(\ln {x_0} = -x_0\)。可知 \(f(x)\) 在 \((0, x_0)\) 上递减，在 \((x_0, +\infin)\) 上递增。\(f(x)_{\min} = f(x_0) = x_0e^{x_0} - x_0 - \ln x_0 - 1 = 1 - 0 - 1 = 0\)。故 \(f(x)\ge 0\)，证毕。

本题的主要思想即为，不显式地求出 \(x_0\)，而是使用 \(x_0\) 进行代换。

例 \(2\)（17 全国 Ⅱ）：已知函数 \(f(x) = ax^2 - ax - x\ln x\)，且 \(f(x)\ge 0\)。

（1）求 \(a\)；

（2）证明：\(f(x)\) 存在唯一极大值点 \(x_0\) 且 \(e^{-2}< f(x_0) < 2^{-2}\)。

首先求解第一问。\(f(x) = x(ax - a - \ln x)\)。由于定义域 \(D = \mathbb R_+\)，所以等价于 \(g(x) = ax - a - \ln x\ge 0\)。

由于 \(g(1) = 0\land g(x)\ge 0\)，故 \(g'(1) = 0\)。又 \(g'(x) = a - \dfrac{1}{x}\)，故 \(g'(1) = 0\) 解得 \(a = 1\)。

验证：当 \(x\in(0,1)\) 时，\(g'(x)<0\)，\(g(x)\) 单调递减，当 \(x>1\) 时，\(g'(x) > 0\)，\(g(x)\) 单调递增，故 \(x=1\) 为 \(g(x)\) 极小值点，所以 \(g(x)\ge g(1) = 0\)，综上 \(a = 1\)。

考虑第二问。由（1），\(f(x) = x^2 - x - x\ln x\)，\(f'(x) = 2x - 2 - \ln x\)。求出 \(f''(x) = 2 - \dfrac{1}{x}\) 知 \(f'(x)\) 在 \((0,\dfrac 12)\) 上单调减，在 \((\dfrac12, +\infin)\) 上单调增。

而 \(f'(\dfrac 12) = \ln 2 - 1 < 0\)，\(f'(e^{-2}) = 2e^{-2} > 0\)，所以由零点定理，存在 \(x_0\in(e^{-2}, \dfrac12)\) 满足 \(f'(x_0) = 0\)。易知 \(x = x_0\) 为 \(f(x)\) 的唯一极大值点。

\(f(x_0) = x_0^2 - x_0 - x_0\ln x_0\)。而 \(2x_0 - 2 - \ln x_0 = 0\)，所以 \(x_0\ln x_0 = 2x_0^2 - 2x_0\)。

于是 \(f(x_0) = x_0 - x_0^2 = x_0(1 - x_0) < \displaystyle\left(\frac{x_0 + 1 - x_0}{2}\right)^2 = 2^{-2}\)，解决了右半边。

对于左半边，由 \(f(x)\) 在 \((0, \dfrac 12)\) 上单调减知，\(f(x_0) > f(e^{-1}) = e^{-2}\)，证毕。
