CF438E The Child and Binary Tree

Aug 14, 2021
阅读时长: 2 分钟

Description

现有一长度为 \(n\) 的互异正整数序列 \(\{c_1, c_2, \cdots, c_n\}\)。令二叉树的权值都从该序列中产生。对任意 \(1\le s\le m\)，求出权值和恰好为 \(s\) 的不同二叉树的个数，答案对 \(998244353\) 取模。

Solution

考虑一个朴素的 DP。令 \(f_i\) 为权值和为 \(i\) 的二叉树的个数。则有如下转移： \[ \begin{cases} f_0 = 1\\ f_n = \sum_{i = 1}^n[i\in C]\sum_{j = 0}^{n - i}f_jf_{n - i - j}& n > 0 \end{cases} \] 然后设生成函数 \(\displaystyle G(x) = \sum_{i \ge 0}[i\in C]x^i\)，\(F(x) = \displaystyle\sum_{i\ge0} f_ix^i\)。然后我们对于 \(n>0\) 有下面这个式子： \[ f_n = \sum_{i = 1}^n g_i\sum_{j = 0}^{n - i}f_jf_{n - i - j} \] 然后 \(i + j + (n - i - j) = n\)，所以这其实就是三个多项式卷到一起，即 \[ F(x) = 1 + G(x)F^2(x) \] 这个多出来的 \(1\) 是指 \(f_0 = 1\)。

用初中知识算一下发现 \[ F(x) = \frac{1\pm\sqrt{1 - 4G(x)}}{2G(x)} \] 那么问题来了，\(+\) 和 \(-\) 应该取哪个呢？考虑取 \(+\) 的情况，下面运用一个叫分母无理化的小技巧。 \[ \begin{aligned} F(x) &= \frac{1 + \sqrt{1 - 4G(x)}}{2G(x)}\\ &= \frac{4G(x)}{2G(x)(1 - \sqrt{1 - 4G(x)})}\\ &= \frac{2}{1 - \sqrt{1 - 4G(x)}} \end{aligned} \] 然后会惊奇的发现当 \(x = 0\) 时整坨式子将不收敛（分母为 \(0\)）。

取 \(-\) 呢？ \[ \begin{aligned} F(x) &= \frac{1 - \sqrt{1 - 4G(x)}}{2G(x)}\\ &= \frac{2}{1 + \sqrt{1 - 4G(x)}} \end{aligned} \] 开根，求逆，做完了。其思想就是将卷积形式的 dp 套上生成函数然后进行推导。

Implementation

犯困，犯了些憨憨错误。。。

int n, m;
Poly g, one(1, 1);

int main() {
    read(n), read(m);
    g.resize(lim = m + 1);
    FOR(i, 1, n) {
        int c; read(c);
        if (c <= m) g[c] = 1;
    }
    g = 2 * polyInv(one + polySqrt(one - 4 * g));
    FOR(i, 1, m) print(g[i]), putchar('\n');
    return output(), 0;
}
