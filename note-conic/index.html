椭圆

基本性质

第一定义

到顶点 \((-c, 0)\) 和 \((c, 0)\) 的距离之和为常数 \(2a\)（\(a > c\)）的点的轨迹。 \[ \frac{x^2}{a} + \frac{y^2}{b} = 1 \]

\(a > b > 0\)。\(c ^ 2 = a^2 - b^2\)。半长轴为 \(a\)，半短轴为 \(b\)，半焦距为 \(c\)。

离心率 \(e = \dfrac c a\)。

第二定义

到 \(F(c, 0)\) 与到定直线 \(x = \dfrac{a^2}{c}\) 的距离之比为定值 \(\dfrac ca\) 的点的轨迹。

焦准距 \(p = \dfrac{b^2}{c}\)。到焦点与到准线的距离之比为 \(e\)。

第三定义

到定点 \((a, 0)\) 和 \((-a, 0)\) 的斜率乘积为定值 \(-\dfrac{b^2}{a^2}\) 的点的轨迹。

推广一下：对于任意关于原点对称的点 \(A(m, n)\)，\(B(-m, -n)\)，上述也是相同的。

直线与椭圆的位置关系

\(y = kx + m\) 与 \(\dfrac{x^2}{a} + \dfrac{y^2}{b} = 1\) 的位置关系：直接联立看 \(\Delta\)。

直接爆算。傻逼高中数学。

椭圆上求点与 \(l\) 的距离最小考虑找出切线 \(l'//l\)，然后爆算。

弦长问题

从两点间距离公式进行推导：

设 \(y = kx + m\) 与 \(\dfrac{x^2}{a} + \dfrac{y^2}{b} = 1\) 交于 \(A\)、\(B\) 两点。

则 \[ \begin{aligned} d_{A,B} &= \sqrt{(x_A - x_B)^2 + (y_A - y_B)^2}\\ &= \sqrt{(1 + k^2)(x_A - x_B)^2}\\ &= \sqrt{(1 + k^2)}\sqrt{(x_A + x_B)^2 - 4x_Ax_B} \end{aligned} \] 或者还有 \[ \begin{aligned} d_{A,B} &= \sqrt{(x_A - x_B)^2 + (y_A - y_B)^2}\\ &= \sqrt{\left(1 + \frac{1}{k^2}\right)(y_A - y_B)^2}\\ &= \sqrt{\left(1 + \frac{1}{k^2}\right)}\sqrt{(y_A + y_B)^2 - 4y_Ay_B} \end{aligned} \] 考虑将两个方程联立之后得到 \(Ax^2 + Bx + c = 0\)，则由 \[ |x_1 - x_2| = \frac{\sqrt\Delta}{|A|} \] 我们有 \[ d_{A,B} = \sqrt{1 + k^2}\frac{\sqrt\Delta}{|A|} \] 可以用来解决弦长问题。

中点弦问题

已知 \(AB\) 是椭圆 \(\dfrac{x^2}{a} + \dfrac{y^2}{b} = 1\) 的弦，\(M(x_0, y_0)\) 是 \(AB\) 中点。求 \(AB\) 的斜率 \(k\)。

点差法：

  • 点在直线上，联立方程
  • 坐标做差，构造中点坐标与斜率的关系。

\[ \begin{cases} \dfrac{x_A^2}{a^2} + \dfrac{y_A^2}{b^2} = 1\\ \dfrac{x_B^2}{a^2} + \dfrac{y_B^2}{b^2} = 1 \end{cases} \]

然后作个差： \[ \begin{aligned} \frac{1}{a^2}(x_A^2 - x_B^2) + \frac{1}{b^2}(y_A^2 - y_B^2) &= 0\\ \frac{1}{a^2}(x_A + x_B)(x_A - x_B) + \frac{1}{b^2}(y_A + y_B)(y_A - y_B) &= 0\\ \frac{y_A - y_B}{x_A - x_B} &= -\frac{b^2}{a^2}\cdot\frac{x_A + x_B}{y_A + y_B}\\ k_{AB} = -\frac{b^2}{a^2}\cdot\frac{x_0}{y_0} \end{aligned} \] 得到结论 \(k_{AB} = -\dfrac{b^2}{a^2}\cdot\dfrac{x_0}{y_0}\)。实操时，写出 \[ \begin{aligned} &\begin{cases} \dfrac{x_A^2}{a^2} + \dfrac{y_A^2}{b^2} = 1\\ \dfrac{x_B^2}{a^2} + \dfrac{y_B^2}{b^2} = 1 \end{cases}\implies \frac{1}{a^2}(x_A^2 - x_B^2) + \frac{1}{b^2}(y_A^2 - y_B^2) = 0\\ \implies & k_{AB} = -\dfrac{b^2}{a^2}\cdot\dfrac{x_0}{y_0} \end{aligned} \] 即可。

焦点三角形

  • \(C_{\triangle PF_1F_2} = 2a + 2c\)。
  • \(l\) 过 \(F_1\) 交椭圆于 \(A\)、\(B\) 两点，\(C_{\triangle F_2AB} = 4a\)。

设 \(\ang F_1PF_2 = \theta\)，\(\ang PF_1F_2 = \alpha\)，\(\ang PF_2F_1 = \beta\)。

则

  • \(S = b^2\tan\dfrac\theta2\)。（重要）

    Proof：考虑用一下余弦定理： $$

    \[\begin{aligned} (2c)^2 &= |PF_1|^2 + |PF_2|^2 - 2|PF_1||PF_2|\cos\theta\\ |PF_1| + |PF_2| &= 2a\\ 4a^2 &= |PF_1|^2 + |PF_2|^2 + 2|PF_1||PF_2|\\ \end{aligned}\]

    \[ 第一行减去第三行后即得到 \] 4b^2 = 2|PF_1||PF_2| + 2|PF_1||PF_2| \[ 故 \] S = |PF_1||PF_2|=b^2 $$

  • \(e = \dfrac{\sin\theta}{\sin\alpha + \sin\beta}\)。

    Proof：考虑正弦定理。 \[ e = \frac c a = \frac{2c}{2a} = \frac{F_1F_2}{PF_1 + PF_2} = \frac{\sin\theta}{\sin\alpha + \sin\beta} \]

  • \(|PF_1||PF_2|=\dfrac{2b^2}{1 + \cos\theta}\)。证明见上。

焦半径公式

焦半径公式（1）

\[ |PF_1| = a + ex_0, |PF_2| = a - ex_0 \]

短的是减，长的是加。

推导：考虑椭圆的第二定义，设准线方程为 \(x = \dfrac {a^2}c\)，则 \(d = \dfrac{a^2}c \pm x_0\)，而 \(|PF| = ed = a \pm ex_0\)。

通径

即 \(x_0 = c\)，通径长： \[ 2(a - \dfrac{c^2}a) = \dfrac{2b^2}{a} \]

双曲线

基本性质

定义：到两定点 \(F_1\)、\(F_2\) 距离之差的绝对值 \(2a\) 为定值的点的轨迹。其中 \(2a < |F_1F_2|\)。

标准方程： \[ \frac{x^2}{a^2} - \frac{y^2}{b^2} = 1 \] 其中 \(a^2 + b^2 = c^2\)，\(F_1(c, 0)\)，\(F_2(-c, 0)\)。

若 \(2a > |F_1F_2|\)，则其为射线；若不带绝对值，则为双曲线的一支。

焦点在 \(y\) 轴上则为 \[ \frac{y^2}{a^2} - \frac{x^2}{b^2} = 1 \] 谁前面的系数是正的，焦点就在哪个轴上面。实操中可以设 \(mx^2 + ny^2 = 1(mn < 0)\) 来避免可能的分类讨论。

抛物线

基本性质

定义：到一定直线 \(l\) 与到 \(P\) 距离相等的点的轨迹叫做抛物线。\(\notin l\)
