CF1033G Chip Game

Jun 23, 2022
阅读时长: 3 分钟

Description

Alice 和 Bob 取石子。\(n\) 堆石子，第 \(i\) 堆石子有 \(v_i\) 个。Alice 每次能在一堆里面取 \(a\) 个，Bob能取 \(b\) 个，其中 \(1\le a,b\le m\)，不能操作的输。考虑双方都绝顶聪明时，对于一对选定的 \((a,b)\) 和先后手情况，有

  • Alice 必胜；
  • Bob 必胜；
  • 先手必胜；
  • 后手必胜。

问在所有的 \((a,b)\) 中每类局面各出现多少次，\(1\le n\le 100\)，\(1\le m\le 10^5\)，\(1\le v_i\le 10^{18}\)。

Solution

首先前两个是等价的，可以不管，用总方案数减掉后面两种方案之和即可。

有一个很牛逼的结论：对于所有 \(v_i'\equiv v_i\pmod{(a+b)}\) 构成的局面 \(S'\)，其与所有 \(v_i\) 构成的 \(S\) 是本质相同的。

证明的话考虑到先手取一个之后后手接着取，这样就 \(-(a+b)\) 了。

那么就直接将所有 \(v_i\) 对 \((a+b)\) 取模，假设 \(a\le b\) （都是对称的，这里只区分一下前后手），则对每个 \(v_i\) 讨论：

  • \(0\le v_i<a\)，这个时候这个点没用；
  • \(a\le v_i<b\)，这个点只能由 A 来取；
  • \(b\le v_i<2a\)，这个点谁取了之后都会变成废点；
  • \(\max(2a, b)\le v_i<a+b\)，如果 A 拿了之后，这个点变为 \(a\) 专属，如果 B 取了就作废。

那么很显然，如果存在一个专属点那么 A 就一定必胜，因为剩下的点 A、B 都能取，A 一直留着专属点，没了再去取即可。

分类讨论下来就是：

  • 有二类数，则 A 必胜。
  • 无二类数：
    • 四类数有大于等于两个，则 A 必胜（随便取一个，然后就有专属点了）；
    • 四类数有一个：
      • \(\sum[v_i\ge b]\) 为奇数则先手必胜。
      • \(\sum[v_i\ge b]\) 为偶数则 A 必胜。
    • 不存在四类数：
      • \(\sum[v_i\ge b]\) 为奇数则先手必胜。
      • \(\sum [v_i\ge b]\) 为偶数则后手必胜。

所以只考虑先手/后手必胜的：

  • 不存在二类数：
    • 四类数有一个，且 \(\sum[v_i\ge b]\) 则先手必胜
    • 不存在四类数：
      • \(\sum[v_i\ge b]\) 为奇数则先手必胜。
      • \(\sum [v_i\ge b]\) 为偶数则后手必胜。

于是 \(O(nm^2)\) 大暴力枚举 \((a,b)\) 即可，但是好像过不去。

考虑先手必胜的情况：

  • 不存在 \(a\le v_i<b\)；
  • \(\sum[\max(2a, b)\le v_i<a+b]\le 1\) 且 \(\sum[v_i\ge b]\) 为奇数。

后手必胜的情况：

  • 不存在 \(a\le v_i<b\)；
  • \(\sum[\max(2a, b)\le v_i<a+b] = 0\) 且 \(\sum[v_i\ge b]\) 为偶数。

那么就枚举 \(a+b\)，然后将 \(v_i\) 按照取模之后大小排序。

那么 \(a,b\) 一定在某个 \((v_j, v_{j + 1}]\) 的区间里，枚举之（记得考虑边界）。然后 \(\sum[v_i\ge b]\) 的奇偶性就有了。设 \(v_j\) 升序排序，则

  • 先手必胜：

    \(\sum[\max(2a, b)\le v_i<a+b]\le 1\)，即 \(\max(2a, b)>v_{n-1}\)，需满足 \(a>\lfloor v_{n-1}/2\rfloor\) 和 \(b>v_{n-1}\) 中的任意一个。

    而 \(b\) 和 \(a\) 在同一区间内，所以 \(b>v_{n-1}\) 的时候 \(a\) 一定 \(>v_{n-1} > \lfloor v_{n-1}/2\rfloor\)，判断 \(a>\lfloor v_{n-1}/2\rfloor\) 即可。

  • 后手必胜：

    \(\sum[\max(2a, b)\le v_i<a+b] = 0\)，即 \(\max(2a, b) > v_n\)。同理只需考虑 \(a>\lfloor v_n/2\rfloor\) 的限制。

最后一定就会框出一个 \(a\) 的范围 \([l,r]\)，其中 \(l = \max(v_j, \lfloor v_n / 2\rfloor) + 1\)，\(r = \min(v_{j+1}, a + b)\)，然后有 \(b\in(v_j, v_{j+1})\)。

现在要去掉 \(a\le b\) 的限制，故 \(a\) 的取值范围最后为 \([\min(l, a + b -r),\max(r, a + b - l)]\)。

Implementation

#include <bits/stdc++.h>
#define FOR(i, a, b) for (int i = (a); i <= (b); ++i)

using namespace std;
using ll = long long;
const int maxn = 105, maxm = 2e5 + 5;
ll v[maxn], ansF, ansL;
int n, m, tmp[maxn];

void solve(int s) {
    FOR(i, 1, n) tmp[i] = v[i] % s;
    tmp[0] = 0, tmp[n + 1] = s - 1;
    sort(tmp, tmp + n + 2);
    FOR(i, 1, n + 1) {
        bool fl = (n + 1 - i) & 1;
        int l = max(tmp[i - 1], tmp[n - 1 + (!fl)] / 2) + 1, r = min(tmp[i], m);
        (fl ? ansF : ansL) += max(0, min(r, s - l) - max(l, s - r) + 1);
    }
    return;
}

int main() {
    cin >> n >> m;
    FOR(i, 1, n) cin >> v[i];
    FOR(x, 2, m << 1) solve(x);
    ll ans1 = (1ll * m * m - ansL - ansF) >> 1;
    cout << ans1 << ' ' << ans1 << ' ' << ansF << ' ' << ansL << endl;
    return 0;
}
